HWP2XLSX v0.4.0
================

한컴오피스 없이 HWP 5.x 및 HWPX 문서를 편집 가능한 XLSX로 변환하는 Windows 프로그램입니다.

사용 방법
---------
1. Hwp2Xlsx.exe를 실행합니다.
2. HWP 또는 HWPX 원본 파일을 선택합니다.
3. 저장할 XLSX 경로를 지정하고 변환합니다.

출력 파일이 이미 있으면 프로그램이 덮어쓰기 여부를 확인합니다. 변환 또는 검증에 실패하더라도 기존 출력 파일은 보존됩니다.

명령줄 사용
-----------
Hwp2Xlsx.exe input.hwp output.xlsx
Hwp2Xlsx.exe input.hwpx output.xlsx
Hwp2Xlsx.exe --overwrite input.hwp output.xlsx

명령줄에서는 기본적으로 기존 결과를 덮어쓰지 않습니다. 교체하려면 --overwrite 또는 -f 옵션을 사용하십시오.

현재 지원
---------
- 페이지별 Excel 시트 생성
- 용지 크기, 방향, 여백과 인쇄 영역
- 본문 및 표 텍스트
- 표 크기, 셀 병합, 배경색과 테두리
- 병합 셀을 포함한 원본 행·열 비율과 셀 안쪽 여백
- 짧은 한 줄 항목의 축소 맞춤과 A4 인쇄 경계 점선
- 글꼴, 크기, 색, 굵게, 기울임, 밑줄, 취소선과 정렬

제한
----
이미지, 도형, 차트, 수식, 머리말·꼬리말과 일부 고급 서식은 아직 변환하지 않습니다. HWP 3.x, 암호화 문서 및 배포용 보호 문서는 지원하지 않습니다.
