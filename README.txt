HWP2XLSX 프로토타입 0.2.0

1. Hwp2Xlsx.exe를 실행합니다.
2. HWP 또는 HWPX 파일을 선택합니다.
3. 저장할 XLSX 파일을 지정하고 변환합니다.

한컴오피스는 필요하지 않습니다. 이 배포본은 Windows x64용 독립 실행형 프로그램입니다.

현재 프로토타입은 텍스트, 기본 표, 셀 병합, 용지 크기 및 명시적 쪽 나눔을 지원합니다.
페이지마다 본문과 표를 하나의 시트에 함께 배치하며 별도의 Data 시트는 만들지 않습니다.
이미지·도형·수식 및 자동 줄바꿈으로 결정되는 정확한 페이지 경계는 향후 개선 대상입니다.
